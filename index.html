Project 3: IoT & Hardware

Smart Irrigation System (ESP32)

  • Hardware Engineering: Circuit design using microcontrollers (ESP32-C6), Raspberry Pi 4B, analog moisture sensors, and power MOSFETs.
  • Absolute Energy Optimization: Implementation of "Deep Sleep" strategies, achieving an ultra-low power consumption of just 36μA for multi-year autonomy (Zero-Maintenance).
  • Wireless Communication: The system integrates Wi-Fi 6 & Bluetooth 5 and is engineered to be fully capable with LoRa, Matter, and Thread for expanded deployments.
  • Cloud & Control: Centralized Flask dashboard for real-time telemetry, bi-directional parameter control, and OTA (Over-the-Air) firmware updates for remote system maintenance.
